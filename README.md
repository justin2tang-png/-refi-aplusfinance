# A Plus Finance Local Site

This is the local-only static rebuild for A Plus Finance.

- No GitHub push is required for this local version.
- Start a local preview from this folder with `python -m http.server 8080`.
- Open `http://localhost:8080/`.

## Deployment Defaults

- Target hosting: Cloudflare Pages
- Build command: none
- Output directory: project root
- Forms: Formspree
- Analytics: GA4

## Before Production

- Confirm final domain and canonical URLs.
- Confirm Formspree endpoint.
- Confirm GA4 measurement ID.
- Confirm disclosure, privacy, contact details, and adviser details.
